# 投研中台 2.0 新机冷启动手册

## 1. Python 版本
建议：`Python 3.11+`（已在 macOS 验证）

## 2. 创建虚拟环境
```bash
python3 -m venv .venv
source .venv/bin/activate
```

## 3. 安装依赖
```bash
pip install --upgrade pip
pip install -r requirements.txt
```

## 4. 安装 Playwright 浏览器
```bash
playwright install
```

## 5. 配置密钥
编辑 `config/secrets.py`：
- `TUSHARE_TOKEN`
- `GEMINI_API_KEY`（如暂时不用可保留占位）

## 6. 启动命令

### 新闻抓取
```bash
python 01_Data_Harvest/playwright_spider.py
```

### A股公告+PDF提取
```bash
python 03_Quant_Data/a_share_fundamental_extractor.py
```

### AkShare+Tushare个股体检
```bash
python 03_Quant_Data/akshare_stock_upgrade.py
```

## 7. 迁移打包前检查
- 不打包 `.venv/`
- 不打包 `.pw_user_data/`
- 不打包本地日志和缓存
- 确保 `config/secrets.py` 不包含真实密钥（可本地保留，不入库）
